# 📌 BIT HIVE (Team.codeHive)
<br>

## 📃 Project Summary
- 비트코인의 실시간 현황 및 차트조회, 모의투자, 커뮤니티를 이용가능한 웹앱  
- 모의투자의를 통한 비트코인시장의 학습 및 수익률 예상, 정보공유
- 가상계좌를 생성하여 실시간 가격을 기준으로 매수와 매도, 실시간 수익률조회 가능
